# 🍔 Food Management App

A full-featured **Food Management Dashboard** built with **React (Vite)**, designed to manage recipes, categories, users, and favorites with a clean and modern UI.

## 🚀 Overview

This application provides a complete system where:

- 🧑‍💼 Admins can manage recipes, categories, and users  
- 🍽 Users can explore and favorite recipes  
- 📊 Dashboard provides an overview of the system  
- 🔐 Secure authentication system with protected routes  

---

## ✨ Features

### 🔐 Authentication System
- Login / Register  
- Forgot / Reset / Change Password  
- Verify Account  
- JWT-based authentication  
- Protected routes  

---

### 📊 Dashboard
- Overview of application data  
- Quick navigation to all modules  

---

### 🍽 Recipes Management
- View all recipes  
- Add new recipes  
- Edit & delete recipes  
- View recipe details  

---

### 🗂 Categories Management
- List all categories  
- Add new categories  
- Delete categories  

---

### 👥 Users Management
- View all users  
- View user details  

---

### ❤️ Favorites
- Add & remove favorite recipes  

---

### 👤 Profile
- View and update user profile  

---

## 🛠️ Tech Stack

- ⚛️ React 19 + Vite  
- 🎨 Bootstrap  
- 🌐 Axios  
- 🔄 React Router  
- 🧾 React Hook Form  
- 🔔 React Toastify  
- 🎯 Context API  
- 🔐 JWT Authentication  
---

## 📁 Project Structure

- Authentication Module  
- Dashboard Module  
- Recipes Module
